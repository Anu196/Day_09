## **Weather Detection & US News (Day-09)**

### **Project Description**

The **Weather Detection & US News** project is an **n8n automation workflow** that retrieves real-time weather information and the latest US news using APIs. The workflow processes the fetched data, optionally generates AI-powered summaries, and automatically sends a well-formatted email containing current weather conditions and top news headlines. This provides users with a convenient daily update in a single message.

---

## **Features**

* 🌤️ Fetches real-time weather information using a Weather API
* 📰 Retrieves the latest US news headlines through a News API
* 🤖 Generates AI-powered summaries for weather and news
* 📧 Automatically sends a formatted email with daily updates
* ⚡ Fully automated workflow built with n8n
* 🔄 Scheduled execution for regular daily reports
* 📋 Combines weather and news into one email
* 📱 Accessible from any device through email
* ⏰ Saves time by delivering important daily information automatically
* 🔒 Secure API integration and automated processing
